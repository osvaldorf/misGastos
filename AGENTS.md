# AGENTS

## Purpose
This repository is currently empty aside from Git metadata. Agents should not make assumptions about the intended application, language, or architecture.

## Guidance for AI agents
- Do not create or modify application code unless the user explicitly provides project files or describes the desired implementation.
- Ask the user for the project scope, target platform, framework, and any existing requirements before starting work.
- If the repository is updated with source code later, reevaluate structure and conventions before making larger changes.

## When contributing
- Confirm whether the repository should contain a backend, frontend, mobile app, CLI, or other type of project.
- Ask for details on build tools, package managers, and preferred languages if they are not already present.
- If asked to bootstrap a project, keep the initial implementation small and request feedback before expanding.
